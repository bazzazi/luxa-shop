# لوکسا | فروشگاه آنلاین مدرن

<div align="center">

**یک رابط فروشگاهی شیک، سریع و واکنش‌گرا برای نمایش محصولات و تجربه خرید روان**

</div>

---

## معرفی پروژه

لوکسا یک وب‌سایت فروشگاهی مدرن با طراحی تیره و مینیمال است که با **HTML، CSS و JavaScript** ساخته شده است.  
این پروژه برای نمایش محصولات، فیلتر کردن آیتم‌ها، جست‌وجوی سریع و مدیریت سبد خرید طراحی شده است.

---

## ویژگی‌ها

- طراحی واکنش‌گرا برای موبایل، تبلت و دسکتاپ
- هدر ثابت و ناوبری روان
- جست‌وجوی داخلی محصولات
- فیلتر بر اساس دسته‌بندی
- نمایش کارت‌های محصول با جزئیات کامل
- سبد خرید تعاملی با امکان افزایش، کاهش و حذف آیتم‌ها
- بخش پیشنهاد ویژه با شمارش معکوس
- بخش مزایا و اعتمادسازی
- بخش نظرات کاربران
- دکمه بازگشت به بالای صفحه
- انیمیشن‌های نرم و رابط کاربری تمیز

---

## تکنولوژی‌ها

- **CSS3**
