## معرفی پروژه

**لوکسا** یک وب‌سایت فروشگاهی مدرن با ظاهر حرفه‌ای و تجربه کاربری روان است که به‌صورت کامل با **HTML5، CSS3 و JavaScript خالص** ساخته شده است.  
این پروژه برای نمایش یک فروشگاه آنلاین زیبا، سبک و قابل‌گسترش طراحی شده و می‌تواند به‌عنوان پایه‌ای مناسب برای توسعه یک فروشگاه واقعی مورد استفاده قرار گیرد.

---

## ویژگی‌ها

- طراحی کاملاً **واکنش‌گرا** برای موبایل، تبلت و دسکتاپ
- رابط کاربری مدرن با چیدمان RTL و فونت فارسی
- هدر ثابت و منوی ناوبری کاربردی
- جست‌وجوی داخلی محصولات
- فیلتر محصولات بر اساس دسته‌بندی
- نمایش کارت محصولات همراه با قیمت، تخفیف و امتیاز
- سبد خرید تعاملی با امکان افزایش/کاهش تعداد و حذف آیتم
- شمارش معکوس پیشنهاد ویژه
- بخش دسته‌بندی‌ها، مزایا و نظرات مشتریان
- دکمه بازگشت به بالای صفحه
- انیمیشن‌ها و افکت‌های نرم برای تجربه کاربری بهتر

## اجرای پروژه

### روش ساده
فایل `index.html` را مستقیم در مرورگر باز کن.

### روش پیشنهادی برای توسعه
اگر می‌خواهی پروژه را با یک سرور محلی اجرا کنی، داخل پوشه پروژه یکی از دستورات زیر را اجرا کن:

```bash
python -m http.server 8000
```

سپس در مرورگر این آدرس را باز کن:

```text
http://localhost:8000
```

## انتشار آنلاین روی GitHub Pages

برای گذاشتن **Live Demo** روی GitHub Pages:

1. یک Repository جدید در GitHub بساز.
2. فایل‌های پروژه را داخل مخزن قرار بده و `push` کن.
3. مطمئن شو فایل `index.html` در ریشه پروژه قرار دارد.
4. به بخش **Settings** مخزن برو.
5. از منوی سمت چپ وارد **Pages** شو.
6. در قسمت **Build and deployment**:
   - گزینه **Source** را روی **Deploy from a branch** بگذار.
   - شاخه `main` یا `master` را انتخاب کن.
   - پوشه را روی `/ (root)` قرار بده.
7. روی **Save** کلیک کن.
8. بعد از چند لحظه، GitHub یک آدرس عمومی برای دمو می‌سازد.

نمونه آدرس:

```text
https://username.github.io/repository-name/
```
